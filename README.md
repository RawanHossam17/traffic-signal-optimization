# traffic-signal-optimization# Traffic Signal Optimization using PSO and Hybrid Approach

This project applies Particle Swarm Optimization (PSO) and a Hybrid approach (PSO + Genetic Algorithm) to optimize traffic signal timings.

## Features
- Traffic simulation with multiple intersections
- PSO implementation
- Hybrid PSO-GA approach
- Performance comparison (30 runs)
- Visualization using graphs

## How to Run

Run main:
  python main.py

Run experiments:
  python -m experiments.run_30


Plot results:
   python -m experiments.plot

GUI:
  py -m streamlit run streamlit_app.py
